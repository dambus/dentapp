# DentApp Project Documentation

This folder contains product, discovery, technical, compliance, execution, and Codex documentation for the DentApp project.

## Main folders

- 00_project - project vision, decisions, glossary, open questions
- 01_discovery - interviews, workflow observation, pilot feedback
- 02_product - MVP scope, roadmap, roles, backlog
- 03_domain - dental domain models and terminology
- 04_ux_ui - UX, UI, screen maps, design system
- 05_technical - architecture, database, storage, auth, deployment
- 06_compliance - legal, data protection, audit, retention, export
- 07_execution - implementation roadmap, tasks, progress
- 08_codex - dedicated instructions for Codex/Cursor
